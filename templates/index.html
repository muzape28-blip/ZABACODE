⚡ ZABACODE Overview

Zabacode adalah Mobile Python IDE 100% Standalone & Anti-Kapitalis dengan zero-telemetry, didesain untuk berjalan di HP Android (ARMv7 32-bit & ARM64 64-bit).

🚀 Fitur Utama & Solusi

  • Deterministic Monaco Cursor Alignment: Kursor menyala bergerak presisi mengikuti baris di Android WebView.
  • Bilingual i18n Engine: Dukungan penuh Bahasa Indonesia dan Bahasa Inggris yang bisa dipindah dalam 1 klik.
  • Functional Plugin Addon System: Auto-Formatter, Snippets, Linter, dan Extended Mobile Symbol Bar.
  • Local Session Auth Token Protection: Mengikat endpoint API lokal dengan token keamanan X-Zabacode-Token acak.
  • Android Keystore Key Encryption: Penyimpanan kunci API terenkripsi aman.
